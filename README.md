PEPFinder2 is used to find phase equilibrium point pairs of chemimal potentials in N-component  polymer systems.<br/>
<h4>STEP 1. Prepare the data file which contains at least two columns of chemical potentials.</h4>
<h4>STEP 2. Set the following parameters in PEPFinder2.m file.</h4>
- data_file: filepath of dataset plain file in which data are separated by comma<br/>
- eps: distance error threshold of chemical potential pairs <br/>
- omega: the least distance of indexes of two chemical potentials entries. 
<h4>STEP 3. Run PEPFinder2.m.</h4>
